## Quando Direcionar para este Painel

- Buracos na rua ou calcada
- Poste apagado, falta de iluminacao
- Poda de arvore, mato alto
- Lixo, entulho na rua
- Esgoto a ceu aberto, bueiro entupido
- Asfalto danificado
- Semaforo quebrado
- Alagamento, enchente
- Terreno abandonado com problemas
- Qualquer problema de infraestrutura urbana

## Quando NAO Direcionar para este Painel

- Problemas DENTRO do imovel do cidadao -> Regularizacao
- Conta de agua/luz -> Regularizacao
- Seguranca publica (assaltos no local) -> Seguranca
