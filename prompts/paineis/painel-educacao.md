# Painel: Educacao

## Campo: Nome do Painel no Helena

```
educacao
```

## Campo: Descricao para o Agente

Copie no campo "Descricao para o Agente" no admin:

```
Demandas de educacao: vagas em creche, escola, matricula, EMEI, EMEF, CEI, transferencia escolar, bolsas de estudo, merenda, transporte escolar, uniforme, material escolar, curso profissionalizante.

Palavras-chave: creche, vaga, escola, matricula, EMEI, EMEF, CEI, transferencia, bolsa, merenda, transporte escolar, uniforme, material escolar, curso.

IMPORTANTE: NAO explique como funciona o processo de matricula ou transferencia. Apenas colete os dados e transfira.
```

## Campo: Requisitos Pre-Transferencia (opcional)

```
Antes de transferir, pergunte:
- E para crianca ou adulto?
- Qual a idade do aluno?
- Qual tipo de vaga precisa? (creche, EMEI, EMEF, etc.)
- Qual regiao/bairro de preferencia?
```

## Quando Direcionar para este Painel

- Vagas em creche, EMEI, EMEF, CEI
- Matricula escolar
- Transferencia de escola
- Bolsas de estudo
- Merenda escolar
- Transporte escolar
- Uniforme e material escolar
- Cursos profissionalizantes

## Quando NAO Direcionar para este Painel

- Cursos universitarios / faculdade -> Atendimento Geral
- Problemas de infraestrutura na escola -> Zeladoria
