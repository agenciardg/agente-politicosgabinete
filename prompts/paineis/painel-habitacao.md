# Painel: Habitacao

## Campo: Nome do Painel no Helena

```
habitacao
```

## Campo: Descricao para o Agente

Copie no campo "Descricao para o Agente" no admin:

```
Demandas de moradia e habitacao: programas habitacionais, COHAB, SEHAB, cadastro habitacional, moradia popular, aluguel social, area de risco, desabamento, interdito, remocao, cortico, ocupacao.

Palavras-chave: moradia, casa propria, COHAB, SEHAB, cadastro habitacional, programa habitacional, aluguel social, area de risco, desabamento, interdito, remocao, cortico.

IMPORTANTE: NAO confirmar ajuda com programas federais (Minha Casa Minha Vida). NAO explicar procedimentos da COHAB ou SEHAB. Apenas colete os dados e transfira.
```

## Campo: Requisitos Pre-Transferencia (opcional)

```
Antes de transferir, pergunte:
- Qual a situacao de moradia atual? (aluguel, rua, area de risco, etc.)
- Ja tem cadastro em algum programa habitacional?
- Quantas pessoas moram na residencia?
```

## Quando Direcionar para este Painel

- Programas habitacionais municipais (COHAB, SEHAB)
- Cadastro habitacional
- Moradia popular
- Aluguel social
- Area de risco, desabamento
- Imovel interditado
- Remocao de moradia
- Cortico, ocupacao

## Quando NAO Direcionar para este Painel

- Minha Casa Minha Vida (programa federal) -> informar que nao atende programas federais
- Regularizacao de imovel (escritura, IPTU) -> Regularizacao
- Conta de agua/luz do imovel -> Regularizacao
