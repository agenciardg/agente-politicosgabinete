# Painel: ONG

## Campo: Nome do Painel no Helena

```
ong
```

## Campo: Descricao para o Agente

Copie no campo "Descricao para o Agente" no admin:

```
Demandas de ONGs e terceiro setor: criacao de ONG, orientacao para ONGs existentes, parcerias com poder publico, projetos sociais, editais, emendas parlamentares para ONGs, prestacao de contas, CNPJ de ONG.

Palavras-chave: ONG, organizacao, terceiro setor, projeto social, parceria, edital, emenda, prestacao de contas, CNPJ, associacao, instituto.

IMPORTANTE: NAO explique procedimentos de criacao de ONG ou legislacao do terceiro setor. Apenas colete os dados e transfira.
```

## Campo: Requisitos Pre-Transferencia (opcional)

```
Antes de transferir, pergunte:
- Ja possui ONG registrada ou quer criar uma?
- Qual a area de atuacao da ONG?
- O que precisa especificamente? (orientacao, parceria, edital, etc.)
```

## Campos Customizados Sugeridos

| Campo Helena | fill_type | Instrucao de Coleta |
|-------------|-----------|-------------------|
| Data e Horario | auto | (automatico) |
| Descricao Manifestacao | auto | (automatico) |
| Tipo de Demanda | auto | (automatico) |
| Nome Completo | contact | (do cadastro) |
| Bairro | contact | (do cadastro) |
| Nome da ONG | collect | Pergunte o nome da ONG (ou se quer criar uma) |
| Area de Atuacao | collect | Pergunte qual a area de atuacao da ONG |
| Tipo de Necessidade | collect | Pergunte o que precisa (orientacao, parceria, edital, etc.) |

## Quando Direcionar para este Painel

- Criacao de ONG, associacao, instituto
- Orientacao para ONGs existentes
- Parcerias com poder publico
- Editais para terceiro setor
- Emendas parlamentares para ONGs
- Prestacao de contas
- Projetos sociais institucionais

## Quando NAO Direcionar para este Painel

- Projeto social pessoal (nao institucional) -> Servico Social
- Voluntariado individual -> Atendimento Geral
