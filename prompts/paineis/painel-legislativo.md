# Painel: Legislativo

## Campo: Nome do Painel no Helena

```
legislativo
```

## Campo: Descricao para o Agente

Copie no campo "Descricao para o Agente" no admin:

```
Demandas legislativas: projetos de lei, propostas, atuacao do politico, tramitacao, votacoes, indicacoes, requerimentos, mocoes, audiencias publicas, comissoes.

Palavras-chave: projeto de lei, proposta, legislacao, votacao, tramitacao, indicacao, requerimento, mocao, audiencia publica, comissao.

IMPORTANTE: NUNCA inventar como o politico votou ou quais projetos apresentou. NUNCA inventar numeros de projetos de lei. Se nao souber, diga que o assessor podera informar. Apenas colete a duvida e transfira.
```

## Campo: Requisitos Pre-Transferencia (opcional)

```
Antes de transferir, pergunte:
- Qual o assunto legislativo de interesse?
- E sobre um projeto especifico ou uma sugestao?
```

## Campos Customizados Sugeridos

| Campo Helena | fill_type | Instrucao de Coleta |
|-------------|-----------|-------------------|
| Data e Horario | auto | (automatico) |
| Descricao Manifestacao | auto | (automatico) |
| Tipo de Demanda | auto | (automatico) |
| Nome Completo | contact | (do cadastro) |
| Bairro | contact | (do cadastro) |
| Assunto Legislativo | collect | Pergunte qual o assunto legislativo de interesse |
| Tipo (consulta/sugestao) | collect | Pergunte se e sobre um projeto especifico ou uma sugestao ao politico |

## Quando Direcionar para este Painel

- Perguntas sobre projetos de lei
- Sugestoes de projetos ou propostas
- Atuacao legislativa do politico
- Tramitacao de projetos
- Audiencias publicas, comissoes
- Indicacoes, requerimentos, mocoes

## Quando NAO Direcionar para este Painel

- Demandas praticas (buraco, saude, escola) -> painel especifico
- Perguntas sobre outros politicos -> informar que so atende o politico do gabinete
